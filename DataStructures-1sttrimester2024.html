Course Overview

This course provides theoretical and practical knowledge of fundamental data structures. Topics covered include Search linked-list, stacks, queues, lists, trees, hash tables and an overview of Java Collections. These data structures are explained using basic searching techniques with brief overview of recursion and memory management. The course also explores the implementation of a range of data structures in the Java programming language. The knowledge and practice of these structures are of utmost importance. It will make the students able to organize, represent and manipulate the data, which is central to computing.

Learning Outcomes

On successful completion of this course students will be able to:
  • Explain the key concepts of data structures and how they operate including Linked Lists, stack, queues, list, trees, heaps, hash tables.
  • Choose the most appropriate data structure and for a given problem.
  • • Build and use these data structures in code.
